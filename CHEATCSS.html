What is CSS?

CSS stands for Cascading StyleSheet. This allows developers to create rules that specify how the content of an HTML element should appear on the web page.

CSS Syntax

  • CSS works by connecting rules with HTML elements.
  • These rules dictate how the content is displayed by the browser
  • A single CSS rule is indicated by the opening and closing curly braces {} and contains 2 parts

    • Selectors and Declarations

  • Selectors indicate which element the rule will apply to. A single rule can apply to many elements by separating the selectors by a comma.
  • Declarations indicate how the element will be styled. All declarations must end with a semicolon ( ; ).
  • Element selectors: Selects HTML elements based on their element name.

    • Why use CSS??

  • CSS is used to define styles for web pages, including design, layout and variations in display for different devices and screen sizes.
  • CSS allows HTML to be more content focused by removing the style formatting from HTML page

    • Three ways CSS can be applied to HTML documents.

    External Stylesheets

  • These are the most preferred way of styling HTML documents. Use one files allows a developer to be more efficient.
  • When creating the CSS style sheets the file must be saved with a .css extension. Each HTML page must include a reference to the external style sheet. This is accomplished by using the< link > element.

    • Internal Stylesheets

  • These are applied by placing the CSS rules inside of a < style> element, which is nested within the < head> element. The < style> element has opening and closing tags.
  • This method should only be used if a site is being built with one web page.

    • Inline Stylesheets

  • These are applied to HTML page by adding the style attribute to the element you want to style. The CSS rules will only apply to that one element.
