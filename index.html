HTML structure

ELEMENT -A single component of an HTML document

  • Block Level- elements begin on a new line and take up full width available on the screen
  • Inline- elements render on the same line by default

    • Link Element -To connect our HTML doc with an outside source

  • Most often used to link to external style sheets.
  • This is an empty element and only contains attributes

    • Paragraph Element

  • Allows developers to structure their HTML doc into paragraphs.
  • Has opening and closing tags
  • Paragraph elements are block-level element

    • Divde Element

  • Most useable element in web development because it allows them to separate content and data on a webpage into sections.
  • Has opening and closing tags
  • These elements are block-level elements

    • Image Element

  • Images are added to a webpage using the < img > element Does not have a closing tag. Call an empty element Requires 2 attributes src (soure): points to the location of the image alt (alternative text): provides a description of the image.

    • TAG-Used to mark the start and end of an element

  • Angle Brackets = The <> tells the web browser that this is an HTML tag
  • Names = The name of the tag usually an abbreviation of a word or phrase
